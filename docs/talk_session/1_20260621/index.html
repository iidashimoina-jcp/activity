    感想¶

    • みんながそれぞれ考えていることが違って面白かった。意見を組み合わせてできることもあるので、新しい発想につながる。（光回線がほしい＋飯田に人が来てほしい→通信インフラ改善で移住者増へ）
    • お菓子たくさん食べられて嬉しかった。
    • 自分の視点ではなかった意見がたくさん出てきた。それらが実はつながっているとイメージできた。
    • 今回声をかけて参加にならなかった人もいるが、次回誘いやすい内容だった。
    • 自分自身の要求を考えるいい時間になった。
    • 身近な問題から出発して、県政の政策にもつながっていくことが実感できた。
    • 自分のモヤモヤを話せた。他の人の意見から自分の困りごとを発掘された。それが県政につながっている、県政でも世界平和につながることを知って面白かった。
